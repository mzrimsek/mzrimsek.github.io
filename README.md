Resume Site
======
This is my resume site found at [zrimsek.com](http://zrimsek.com).

## Technology
* Node
* Jade
* SASS
* Gulp
* Bootstrap
* JQuery
* AngularJS

## Build Process
* `npm install`
* `gulp build`
* `node server.js`

## With Docker
* `docker build -t mzrimsek/resume-site .`
* `docker run -p 3000:3000 -d mzrimsek/resume-site`
